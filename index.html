Logic Synthesis & Constraint Setup

Cadence Genus Synopsys Design Compiler SDC Constraints

Logic synthesis translates the register-transfer level (RTL) code into a technology-mapped gate-level netlist. I perform:

  • Logic Synthesis & Optimization: Timing/power/area trade-offs targeting specific frequencies.
  • Constraint Setup: Developing Synopsys Design Constraints (SDC) including clocks, generated clocks, input/output delays, false paths, and multi-cycle paths.
  • Design Implementation: Synthesized critical timing paths for block "RP-TOP" using Genus and high-complexity 400K-cell "TILE" using Design Compiler (DC Shell).

Floorplanning & Power Planning

Macro Placement Power Grid Design IR Drop Compliance

Creating a robust physical layout floorplan to optimize block area and guarantee reliable power distribution. Experience includes:

  • Macro Placement: Placed 6 macros in 'RP-TOP' and 18 macros in 'TILE' hierarchical floorplans to minimize routing congestion and reduce critical wire lengths.
  • Power Grid Design (PG): Built multi-layer PG mesh configurations for power and ground routing across 9 metal layers (M1-M9).
  • IR Drop Optimization: Designed grids to satisfy electromigration (EM) and IR drop limits, ensuring stable supply voltages across all standard cells and macro domains.

Cell Placement & Congestion Mitigation

Standard Cell Placement Congestion Screens Weight Preferences

Positioning standard cells while managing design density and preventing routing congestion. Key practices:

  • Density Optimization: Used partial blockages, density bounds, and cell padding to distribute cells away from high-density hotspots.
  • Path Optimization: Implemented custom SDC path groups and cell weight preferences to prioritize setup timing closure during the placement stage.
  • Congestion Management: Successfully resolved aggressive routing congestion in a 400K+ cell block using strategic density screens.

Clock Tree Synthesis (CTS)

Clock Skew Control Clock Latency Synopsys ICC2

Building a low-skew clock distribution network to supply clock signals to all registers. Tasks include:

  • Balanced Clock Trees: Synthesized clock networks in Synopsys ICC2, using custom buffer/inverter sets to meet latency budgets.
  • Skew & Latency Control: Satisfied skew target constraints across multiple clock domains in "RP-TOP" design.
  • Crosstalk & Noise Control: Applied shielding rules and non-default routing (NDR) rules to critical clock tree trunks.

Global & Detail Routing

9-Layer Stack Pin Access DRC-Clean Closure

Connecting physical signal terminals using the technology metal stack while satisfying design rules. Focus areas:

  • Layer Utilization: Routed signals utilizing 9 metal layers (M1-M9) in 'RP-TOP'.
  • Pin Access & DRC Fixes: Optimized standard cell pin access and solved post-route DRC violations using metal rerouting.
  • Congestion Resolution: Prevented routing shorts and hot-spots in high-density blocks.

Static Timing Analysis & ECO

STA (Setup/Hold) ECO Fixes WNS / TNS Closure

Verifying timing performance across process corners and temperatures. Core capabilities:

  • Multi-Corner STA: Validated setup and hold timing closure across process corners (SS, TT, FF) and voltage/temperature extremes.
  • ECO Implementation: Applied Engineering Change Orders (ECOs) by inserting delay buffers for hold and sizing gates/swapping VT cells for setup optimization.
  • WNS/TNS Closure: Effectively optimized Worst Negative Slack (WNS) and Total Negative Slack (TNS) to zero violations.

Physical Verification & Sign-off

DRC / LVS / ERC Antenna Fixes GDS Tape-out

Final sign-off check to confirm the layout matches the netlist schematic and meets foundry fabrication rules:

  • Design Rule Checking (DRC): Fixed spacing, width, and density rules for final GDS tape-out readiness.
  • Layout Vs Schematic (LVS): Checked connectivity, transistor matching, and shorts/opens to achieve clean LVS status.
  • Antenna Effect Fixes: Solved antenna ratio violations by inserting protection diodes and performing metal layer hops/rerouting.
